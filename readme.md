## Installation

### Automatic Installation (Recommended)
1. Open Sublime Text.
2. Install [Package Control](https://packagecontrol.io/installation) if you haven't already.
3. Press `Ctrl+Shift+P` (Windows) or `Cmd+Shift+P` (Mac) to open the Command Palette.
4. Search for `Package Control: Add Repository` and select it.
5. Enter the URL of this repository: `https://github.com/your-repository-url`.
6. Search for `CiscoSyntax BadWifi` in the Command Palette using `Package Control: Install Package` and install it.

### Manual Installation
1. Navigate to the Sublime Text `Packages` folder:
   - On Windows: `%AppData%\Sublime Text\Packages\User`
   - On Mac: `~/Library/Application Support/Sublime Text/Packages/User`
   - On Linux: `~/.config/sublime-text/Packages/User`
2. Open a terminal and clone the repository into the `User` folder:
   ```bash
   git clone https://github.com/your-repository-url CiscoSyntax-BadWifi
